Multi-Modal Intelligent Chatbot

Welcome to the Multi-Modal Intelligent Chatbot System. This application provides a RESTful API for interacting with AI models, managing documents, and scraping web content.

API Endpoints

Authentication

POST /api/auth/register - Register a new user

POST /api/auth/login - Login and get access token

Documents

POST /api/documents/upload - Upload a document

GET /api/documents - List all documents

Chat

POST /api/chat - Send a message to the AI

Web Scraping

POST /api/scrape - Scrape content from a URL

For more information, please refer to the API documentation.
